## Getting Started

### Prerequisites
- Node.js (v16+)
- npm/yarn

### Installation

```bash
# Clone the repository
git clone [repository-url]

# Navigate to project directory
cd client

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be available at `http://localhost:5173`
